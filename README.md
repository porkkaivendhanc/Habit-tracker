# Habit-tracker
Habit Tracker is a simple web app that helps users add habits, check them daily, and track streaks.
The app stores everything in the browser using LocalStorage, so the data stays even after refreshing the page.

**Features:**

-Add new habits

-Mark habits as completed for the day

-Automatic streak tracking

-Shows total habits, active streaks, and longest streak

-Simple analytics like total completions and average completion

-Modern and clean user interface

**Files included:**

-index.html (main page)

-style.css (design and layout)

-script.js (logic and streak functions)

**How to use:**

-Open index.html in your browser

-Click “Add Habit” to create a habit

-Click “Done” each day to increase your streak

-Your progress updates automatically

All data is saved offline in your browser.
